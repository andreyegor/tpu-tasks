# [Canvas](https://docs.google.com/document/d/1bBBBgZaY8veo9BYY9wOgIZRuNSy4yEGlElp5g5NUJyo/edit?tab=t.0#heading=h.d4cxbb9qq83z)

## Цель

Создать приложение способное отображать двумерное изображение, состоящее из простых фигур.

Что может пригодиться для выполнения:

- Stage
- Scene
- VBox
- Canvas
- GraphicsContext
- FileChooser

## Этап 1. Подготовить сцену

Для начала создадим проект с JavaFX и подготовим разметку. На экране мы должны видеть большое поле для отрисовки нашего изображение, представленное элементом Canvas, и кнопка для выбора файла. Можете использовать fxml для создания разметки, а можете создавать объекты вручную, на ваше усмотрение

## Этап 2. Учимся рисовать

Скопируйте ваши классы геометрических Фигур из лабораторной №9 прошлого семестра, а также связанные с сериализацией классы.

Измените сигнатуру метода отрисовки - в параметрах он должен принимать экземпляр класса GraphicsContext.

Научите каждую фигуру отрисовывать саму себя средствами GraphicsContext.

Обратите особое внимание на Эллипс, у GraphicsContext нет методов позволяющих легко нарисовать правильный эллипс. Не используйте объект javafx.scene.shape.Ellipse, т.к. он является наследником Node, то есть частью разметки, и не предназначен для отрисовки средствами GraphicsContext. Найдите способ изобразить эллипс хотя бы приблизительно.

## Этап 3. Читаем изображение

С помощью Лабораторной работы №9 прошлого семестра создайте минимум два файла, содержащие наборы фигур, которые образуют различные схематичные изображения. Например: домик с окном, дверью, трубой и дымом из нее, но можете дать волю фантазии и изобразить что-то свое. Если потребуется можете добавьте еще типы фигур: кольца, произвольные четырехугольники и пр.

Не забудьте добавить созданный файл с данными в гит.

По нажатию кнопки в нашей разметке открывается окно выбора файла. JavaFX предоставляет удобный инструмент для выбора файлов.
Когда файл будет выбран - прочитайте его и отрисуйте записанные в нем фигуры на Canvas.

## Чеклист

- [ ] Разметка содержит канвас и кнопку открытия файла
- [ ] При клике по кнопке открывается пикер файлов
- [ ] Выбор файла с фигурами очищает канвас и отрисовывает содержимое выбранного файла
- [ ] Последовательное открытие нескольких файлов не оставляет артефактов и не вызывает других проблем
- [ ] Отрисовка каждого типа фигур происходит корректно, согласно его описанию
