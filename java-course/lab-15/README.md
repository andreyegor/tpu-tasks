## Модули

В рамках прошлой работы мы создали полноценный калькулятор. 
Сделаем еще шажок в сторону организации кода и разнесем код на несколько модулей.

Создайте в проекте с калькулятором новый java-модуль. 

Подключите ваш новый модуль к основному модулю проекта как зависимость.

Перенесите в новый модуль все, что связано с описанием выражений и вычислениями.

## Unit-тестирование

Теперь, когда ваши инструменты для вычисления описаны в отдельном модуле, можно в этом же модуле создать несколько Unit-тестов.

Подключите библиотеку junit к вашему модулю, если он не подключен.

Выделите ряд моментов, которые необходимо проверить в работе вашего инструментария для вычислений. 
Создайте под каждый конкретный случай свой тест.

Проверьте уровень покрытия тестами вашего модуля.

Оцените непокрытую тестами части кода, если такие имеются. Обоснуйте почему эту часть кода не имеет смысла покрывать автотестами
